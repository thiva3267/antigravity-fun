# 👐 Hand-Tracking Elastic String Prototype

An interactive WebXR / Antigravity prototype built with Vanilla JavaScript and MediaPipe. 

### 🌟 Features
- Real-time webcam feed with face visibility.
- Tracks two index fingers simultaneously.
- Creates a dynamic "elastic string" between fingers.
- **Physics included:** Line thickness and color tension (Cyan to Neon Pink) change dynamically based on the distance between hands.

### 🛠️ Tech Stack
- HTML5 Canvas
- Vanilla JavaScript
- Google MediaPipe (Hands & Camera Utils)

### 🚀 How to Run
Just open `index.html` in any modern web browser (Chrome/Safari recommended) and allow camera permissions!
